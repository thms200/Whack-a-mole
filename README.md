# Whack-a-mole
- 두더지 게임
  * 총 10개의 두더지가 나옴
  * 두더지를 클릭하면 1초 뒤 새로운 두더지가 나옴
  * 두더지를 클릭하지 않으면 3초 뒤 두더지는 사라짐
